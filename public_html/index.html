Repository project

Function

A repository is where all the re-used and compiled software is stored.

It is basically an FTP server with certain users, groups and a specific directory structure. It's reachable for any client who has the appropriate credits.

Installation

Installation is done by a Python script which creates the underlying directory structure.

Access

By any FTP client which knows the IP and the Port of the repository and can authenticate itself.

Access rights

Clients can only access the protected directory of their firm, ie. client.com's user can only access /protected/com.client/ directory.

Types of software

public

Software which is reachable by anyone who has a login.

protected

Software which is shared between the client and us. This software might be confidential between the client and us. It must not reachable by other clients than the one, whose project is using it.

private

Software which is not to be shared the public nor any client.
